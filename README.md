**Arun Mobiles - IT Security & Compliance Dashboard**

A modern, enterprise-grade IT audit and compliance dashboard designed to monitor, analyze, and manage organizational security frameworks and risk posture in real time.

**Features**

**KPI Overview Cards**

Frameworks Active
Average Compliance Score
Open Security Gaps
Certifications Achieved

**Compliance Framework Tracking**

SOC 2, ISO 27001, PCI DSS, GDPR, NIST CSF, HIPAA
Real-time compliance percentage and control status
Certification status (Certified / In Review / Planning)

**Security Monitoring Modules**

Threat Intelligence with alert indicators
Vulnerability Scan with issue counts
Firewall Rules monitoring
Access Control tracking

**Risk & Gap Analysis
**
Identify open control gaps
Risk scoring based on severity
Visual indicators for High / Medium / Low risks

**Audit Reporting System
**
Detailed audit reports with findings
Categorized issues: Critical, Major, Minor
Recommendations for compliance improvement

**Real-Time Dashboard
**
Live system status updates
Compliance trends across frameworks
Interactive UI for quick insights

**Settings & Controls
**
User access management
Alert and notification toggles
Policy configuration and retention settings
🛠 Tech Stack
HTML5, CSS3, JavaScript (Vanilla)
Chart.js (for analytics visualization)
Responsive UI with modern dark theme
REST API integration (optional for real-time data)
